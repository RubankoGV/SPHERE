# Сборка .ipa файла для AltStore

## ⚠️ Важно: Требования

Для сборки iOS приложения **обязательно** нужен:
- **macOS** (MacBook, iMac, Mac mini и т.д.)
- **Xcode** (бесплатно из App Store)
- **Apple Developer Account** (бесплатный для личного использования)

Без этих компонентов **невозможно** скомпилировать iOS приложение.

## Варианты решения

### Вариант 1: GitHub Actions (Автоматическая сборка)

Я создал GitHub Actions workflow, который будет автоматически собирать приложение при каждом push.

**Как использовать:**
1. Убедитесь, что код загружен на GitHub
2. Перейдите в раздел **Actions** вашего репозитория
3. Запустите workflow **"Build iOS App for AltStore"**
4. После завершения скачайте .ipa файл из артефактов

**Ограничения:**
- GitHub Actions требует правильную настройку проекта Xcode
- Может потребоваться дополнительная настройка подписи

### Вариант 2: Использовать Mac (Рекомендуется)

Если у вас есть доступ к Mac:

#### Шаг 1: Откройте проект в Xcode

```bash
# На Mac, клонируйте репозиторий
git clone https://github.com/RubankoGV/SPHERE.git
cd SPHERE
```

#### Шаг 2: Создайте Xcode проект

1. Откройте Xcode
2. File → New → Project
3. Выберите **iOS** → **App**
4. Заполните:
   - Product Name: `SPHERE`
   - Interface: `SwiftUI`
   - Language: `Swift`
5. Сохраните проект

#### Шаг 3: Добавьте файлы

1. Перетащите всю папку `SPHERE` в проект Xcode
2. Убедитесь, что все файлы добавлены в Target

#### Шаг 4: Настройте подпись

1. Выберите проект в навигаторе
2. Выберите Target **SPHERE**
3. Перейдите в **Signing & Capabilities**
4. Выберите ваш Team (или создайте бесплатный Apple ID)
5. Xcode автоматически создаст Provisioning Profile

#### Шаг 5: Соберите для устройства

1. Подключите iPhone к Mac
2. Выберите ваше устройство в списке устройств
3. Нажмите **⌘R** или кнопку **Run**

#### Шаг 6: Экспортируйте для AltStore

1. Product → Archive
2. После завершения откроется окно Organizer
3. Выберите архив и нажмите **Distribute App**
4. Выберите **Ad Hoc** (для AltStore)
5. Выберите ваше устройство
6. Экспортируйте .ipa файл

### Вариант 3: Облачные сервисы

Можно использовать облачные Mac сервисы:
- **MacStadium** (платный)
- **MacinCloud** (платный)
- **AWS EC2 Mac instances** (платный)

### Вариант 4: Использовать готовый .ipa (если есть)

Если кто-то уже собрал приложение, можно использовать готовый .ipa файл.

## Установка через AltStore

После получения .ipa файла:

1. Установите **AltStore** на iPhone
2. Откройте AltStore
3. Нажмите **+** → **Install .ipa**
4. Выберите файл SPHERE.ipa
5. Дождитесь установки

## Альтернатива: TestFlight

Если у вас есть Apple Developer Account ($99/год):
1. Загрузите приложение в App Store Connect
2. Добавьте в TestFlight
3. Установите через TestFlight (проще, чем AltStore)
