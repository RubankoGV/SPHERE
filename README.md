# SPHERE - Персональный трекер развития

## Описание

SPHERE — это премиальное iOS приложение для отслеживания развития во всех сферах жизни. Приложение использует игровую механику и визуализацию прогресса в виде многоугольника для мотивации пользователя к ежедневному росту.

## Основные функции

### 14 сфер жизни
1. Здоровье
2. Красота
3. Учёба
4. Ментальное здоровье
5. Спорт
6. Личная жизнь
7. Финансы
8. Саморазвитие
9. Целеустремлённость
10. Продуктивность
11. Карьера
12. Навыки общения
13. Организованность
14. Дисциплина

### Система челленджей
- Более 140 уникальных челленджей (минимум 10 на каждую сферу)
- 4 уровня сложности: Начинающий, Средний, Продвинутый, Эксперт
- Ежедневные и долгосрочные челленджи
- Адаптация под уровень пользователя

### Система прогресса
- Визуализация в виде многоугольника (14 граней)
- Уровни для каждой сферы
- Система опыта (XP)
- Streak (серии дней)
- Общий уровень пользователя

### Интеграции
- Apple Health (шаги, калории, сон)
- Calendar (события)
- Reminders (напоминания)
- UserNotifications (уведомления)

### Социальные функции
- Таблица лидеров
- Друзья
- Сравнение прогресса

### Дизайн
- Стеклянный дизайн (Glassmorphism) в стиле iOS 26
- Адаптивная тема (светлая/тёмная)
- Плавные анимации
- Премиальный UI/UX

## Требования

- iOS 16.0+
- Xcode 14.0+
- Swift 5.7+

## Установка

1. Откройте проект в Xcode
2. Выберите целевое устройство или симулятор
3. Нажмите Run (⌘R)

## Использование

### Первый запуск
1. При первом запуске пользователь проходит онбординг:
   - Ввод имени
   - Выбор возраста
   - Выбор интересующих сфер
   - Определение уровня опыта
   - Постановка целей

### Основной функционал
- **Главный экран**: Обзор всех сфер и многоугольник прогресса
- **Челленджи**: Просмотр и выполнение челленджей
- **Прогресс**: Детальная статистика по всем сферам
- **Профиль**: Настройки и статистика пользователя

### Выполнение челленджа
1. Выберите сферу или откройте список всех челленджей
2. Найдите интересующий челлендж
3. Нажмите на кнопку выполнения
4. Получите XP и обновите прогресс

## Разработка

### Добавление нового челленджа
Отредактируйте `ChallengeLibrary.getChallenges(for:userLevel:)` в файле `Challenge.swift`

### Настройка цветов
Измените расширение `Color` в файле `PolygonProgressView.swift`

### Добавление новой сферы
1. Добавьте новый case в enum `LifeSphere`
2. Добавьте соответствующие свойства (icon, color, description)
3. Обновите `ChallengeLibrary` для новой сферы
